Teorema 3.3.15.

(Generalização do 1º Lema de Kaplansky) O número de subconjuntos com \(p\) elementos de \(\{1,2,3,\cdots,n\}\) de modo que entre cada dois elementos escolhidos para o subconjunto haja, no conjunto, pelo menos \(r\) elementos não escolhidos para o subconjunto é:

\begin{equation*} KG(n,p,r)=C_{n-(p-1)r}^p. \end{equation*}
Demonstração.
