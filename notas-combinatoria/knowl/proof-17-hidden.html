Representando os \(p\) elementos que pertencerão a este subconjunto com o sinal de \((+)\) e os \(n-p\) elementos não selecionados com o sinal de \((-)\text{,}\) precisamos organizá-los de forma que entre dois sinais de \((+)\) haja pelo menos \(r\) sinais de \((-)\text{.}\) Formando uma fila com os sinais de \((+)\) ficamos com \(p+1\) espaços para colocar os sinais de \((-)\text{,}\) contando com os lugares: antes do primeiro \((+)\) e depois do último \((+)\text{.}\)

Precisamos contar quantos sinais de \((-)\) vão ficar entre os sinais de \((+)\text{,}\) satisfazendo que antes do primeiro ou do último \((+)\) haja qualquer quantidade de sinais de \((-)\) e entre dois sinais de \((+)\text{,}\) haja pelo menos \(r\) sinais de \((-)\text{.}\) Como a quantidade de sinais \((-)\) disponíveis é \(n-p\text{,}\) o número de soluções desse problema é o mesmo que o número de soluções em números inteiros, da equação:

\begin{equation*} x_1+x_2+\cdots+x_{p+1} = n-p, \text{ com } x_1\geq 0, x_{p+1}\geq 0 \text{ e } x_2, \ldots, x_p\geq r. \end{equation*}

Definindo \(x_2 = r+y_2, \ldots, x_p = r+y_p\text{,}\) e substituindo na equação anterior, ficamos com

\begin{equation*} x_1 + r+y_2 + r+y_3 + \cdots + r+y_p + x_{p+1} = n-p. \end{equation*}

Ou seja,

\begin{equation*} x_1 + y_2 + y_3 + \cdots + y_p + x_{p+1} = n-p - (p-1)\times r. \end{equation*}

O número de soluções desta equação é

\begin{equation*} CR_{p+1}^{n-p-(p-1)r} = C_{n-(p-1)r}^p. \end{equation*}
