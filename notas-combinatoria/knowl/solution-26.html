Solução 2.6.9.1.

Observe que uma possibilidade seria calcular o número de soluções de cada um dos casos:

\begin{equation} \begin{cases} x_1 + x_2 + x_3 + x_4 = 0,\\ x_1 + x_2 + x_3 + x_4 = 1\\ \vdots \\ x_1 + x_2 + x_3 + x_4 = 50.\end{cases}\tag{2.6.3} \end{equation}

A soma do número de soluções de cada um dos casos é a resposta, no entanto, é inviável fazer tal cálculo. Felizmente temos outra forma de resolver este problema.

Observe que existe uma bijeção entre o conjunto das soluções de (2.6.2) com o conjunto das soluções da equação (2.6.4):

\begin{equation} x_1 + x_2 + x_3 + x_4 + y = 50.\tag{2.6.4} \end{equation}

Para entender a bijeção, observe o seguinte. Somando \(50-i, i\in \{0, 1, 2, \ldots, 50\}\text{,}\) nos dois lados da igualdade, na linha \(i, i\in \{0, 1, 2, \ldots, 50\}\) de (2.6.3), não mudamos absolutamente nada e ficamos com:

\begin{equation} \begin{cases} x_1 + x_2 + x_3 + x_4 + 50 = 0 + 50,\\ x_1 + x_2 + x_3 + x_4 + 49 = 1 +49 \\ \vdots \\ x_1 + x_2 + x_3 + x_4 + 0 = 50+0.\end{cases}\tag{2.6.5} \end{equation}

Cada solução da linha \(i\text{,}\) é uma solução de (2.6.4) com o valor de \(y\) igual a \(50-i\text{.}\) E para cada \(y\in \{0, 1, 2, \ldots, 50\}\) a solução de (2.6.4) vai ser a solução da linha \(y\) de (2.6.3) .

Portanto a resposta é o número de solução da equação (2.6.4) que é dado por:

\begin{equation*} CR_5^{50} = 316251. \end{equation*}
