Solução 1.7.8.1.

Vamos contar, como se os números não estivessem organizados de forma circular. Depois vamos adicionar as peças que ficam faltando.

Temos que escolher os valores de cada um dos 3 lados de cada peça do trominó. Como os valores vão de 1 até 6 e são 3 lados, o número de peças do trominó (sem contar as permutações circulares) para \(n=6\) é o número de soluções inteiras não negativas da equação:

\begin{equation*} x_1 + x_2 + x_3 + x_4 + x_ 5 + x_ 6 = 3. \end{equation*}

Agora precisamos contar as peças que estão faltando. As peças com três números diferentes, podem ser feitas de duas formas, pois o número de permutações circulares com 3 elementos distintos é 2. Assim, temos

\begin{equation*} 2\times C_6^3 =40 \end{equation*}

peças com 3 números distintos. Como já contamos elas uma vez, na contagem anterior, a quantidade total de peças é:

\begin{equation*} CR_6^{3} + C_6^3 = 56+20 = 76. \end{equation*}
